Чтобы запустить наш проект нужно:
1. Установить компилятор CoffesScript. Например WebStorm (Он. по-моему, самый удобный, поэтому я буду пояснять на основе него.
2. Далее нужно настроить WebStorm. Нажимем File>Settings. Ищем File Watcher и ставим галочку у строки CoffeeScript, а затем нажать справа "Редактировать", либо просто два раза тыркнуть на строке. В открывшемся окне (Edit Watcher) в графе Program нужно указать путь к node.exe из нашего проекта. В графе arguments указать путь к папку coffee (Если в пути есть пробелы, нужно записать в ковычках).
